Page d'aide (Thalassa)

1. Description

Le projet est une mini application de morphing entre un, deux ou plusieurs dessins crées à l'avance à l'aide de l'éditeur de dessin.

Le premier canvas permet d'afficher le morphing des dessins et le deuxième est l'éditeur qui permet de créer et d'éditer un dessin.

2. Les differents manipulation

Editeur de dessin

  • Un clic gauche de la souris sur le canvas de l'éditeur permet de commencer ou de continuer une polyligne.
  • Shift+Clic gauche de la souris permet de commencer un nouvelle polyligne.
  • Tout autre clic différent du clic gauche sur la souris permet de revenir en arrière.
  • Ctrl+G pour la symetrie verticale du dessin sur l'éditeur.
  • Ctrl+H pour la symetrie horizontale du dessin sur l'éditeur.
  • Cliquez sur le bouton IMPORT pour importer le dessin dans le model (Il faut que le dessin sur l'éditeur et le contenu du text-area concordent).
  • Vous pouvez aussi modifier le dessin sur l'éditeur en écrivant des coordonnées dans le text-area et en appuyant sur DISPLAY AREA.
  • Choisissez un model à exporter et appuyez sur le bouton EXPORT pour exporter le model sur l'éditeur de dessin.
  • Vous pouvez changer de couleur à chaque segment en choisissant un couleur avant de le dessiner sur l'éditeur.
  • Si vous utilisez le text-area pour dessiner la couleur sera appliqué à tous les segments.

Morphing

  • Un clic sur le bouton START permet de commencer le morphing des differents dessins crées.
  • Ctrl+flèche gauche ou un clic sur le bouton NEXT permet de passer au dessin suivant.
  • Ctrl+flèche droit ou un clic sur le bouton LAST permet de passer au dessin precedent.
  • Ctrl+flèche haut ou un clic sur le bouton MIRROR-V permet de passer à la symetrie verticale du dessin courant.
  • Ctrl+flèche bas ou un clic sur le bouton MIRROR-H permet de passer à la symetrie horizontale du dessin courant.
  • Un clic sur le bouton STOP permet de stopper le morphing.
